# create-rsuite-app

## 用法
#### 安装

```bash
# 确保全局安装
npm install -g yo
npm install -g generator-create-rsuite-app
```

#### 创建项目
```bash
# 创建并进入项目文件夹
mkdir your-project && cd your-project

// 使用脚手架安装项目
yo create-rsuite-app
```

#### 运行项目
等待 npm install 安装完成后，执行下面命令，打开 `localhost:3002` 即可运行项目。
```bash
npm run dev
```
